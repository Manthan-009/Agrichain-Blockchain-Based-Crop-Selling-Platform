# 🌾 Agrichain – Blockchain Based Crop Selling Platform

Agrichain is a blockchain-based crop selling platform developed to create a transparent, secure, and decentralized marketplace for farmers and buyers. The platform enables farmers to sell agricultural products directly to customers without intermediaries, ensuring fair pricing and better profits.

This project integrates blockchain technology to maintain secure and immutable transaction records, improving trust and transparency in the agricultural supply chain. Buyers can explore available crops, place orders, and perform secure transactions, while farmers can manage crop listings and monitor sales efficiently.

---

## 🚀 Features

- 👨‍🌾 Farmer & Buyer Authentication  
- 🌱 Crop Listing and Management  
- 🔐 Secure Blockchain Transactions  
- 📦 Order Management System  
- 💳 MetaMask Wallet Integration  
- ⛓️ Smart Contract Support  
- 📊 Transparent Supply Chain Tracking  
- 📱 Responsive User Interface

## 🎯 Objective

The main objective of Agrichain is to empower farmers by providing a decentralized digital marketplace where they can directly connect with buyers, reduce dependency on intermediaries, and ensure transparency in crop trading using blockchain technology.
